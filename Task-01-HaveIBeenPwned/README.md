# Task 01 — HaveIBeenPwned Breach Check

## Objective
Check if personal email has appeared in any known data breaches.

## Tool Used
- [HaveIBeenPwned](https://haveibeenpwned.com)

## What I Did
Ran my primary email through HaveIBeenPwned — a tool used 
by cybersecurity professionals to check for known data breaches.

## Findings
- Total breaches found: 0
- Result: Clean — no pwnage detected

## Screenshot
![HIBP Result](../assets/task01-hibp-result.png.com)

## Key Takeaway
A clean result doesn't mean you're fully safe. HIBP only 
tracks publicly disclosed breaches. Unknown leaks exist 
that never make it into the database. This is why breach 
monitoring and threat intelligence matter in SOC work.
